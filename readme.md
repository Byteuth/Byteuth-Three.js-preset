three.js starter boilerplate with very basic functionality.

adds: 
- canvas, 
- scene, 
- camera,
- rendered,
- basicMesh, 
- axeshelper, 
- clock
- orbitalControls

basic listerners:
- resize, dblclick, mousemove

``` bash
npm install

npm run dev
npm run build
```
